Evaluation of The Model:


To evaluate the model's performance, the Root Mean Square Error (RMSE) is calculated. RMSE measures the difference between the predicted and actual ratings, providing insight into the accuracy of the model's predictions.


Conclusion


In this report, we implemented Model-based Collaborative Filtering using Singular Value Decomposition to make product recommendations. The decomposition allowed us to capture latent features that drive user preferences. The recommendation function provided personalized suggestions to users based on their predicted preferences. The evaluation through RMSE indicated the model's predictive performance, with a RMSE value of approximately 0.0137, indicating reasonably accurate predictions.

This model shall be refined and integrated with a UI to provide users with personalized product recommendations based on their historical interactions and preferences.
